# API reference

`airflow_cron` re-exports its Airflow configuration and factory together with
the underlying `cron-pydantic` models.

| [`CronAirflowConfiguration`](_build/airflow_cron.CronAirflowConfiguration.md#airflow_cron.CronAirflowConfiguration)   | Cron configuration plus defaults for generated Airflow DAGs and tasks.                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [`create_dags`](_build/airflow_cron.create_dags.md#airflow_cron.create_dags)(config)                                  | Convert enabled user-crontab jobs into airflow-pydantic DAG models.                                                                                                                                                                                                        |
| [`CronSchedule`](_build/airflow_cron.CronSchedule.md#airflow_cron.CronSchedule)                                       | Five-field Vixie/Cronie schedule.                                                                                                                                                                                                                                          |
| [`CronJobConfiguration`](_build/airflow_cron.CronJobConfiguration.md#airflow_cron.CronJobConfiguration)               | Command and schedule for one crontab entry.                                                                                                                                                                                                                                |
| [`CronConfiguration`](_build/airflow_cron.CronConfiguration.md#airflow_cron.CronConfiguration)                        | Environment and named jobs rendered as one crontab file.                                                                                                                                                                                                                   |
| [`SpecialSchedule`](_build/airflow_cron.SpecialSchedule.md#airflow_cron.SpecialSchedule)                              | alias of `Literal`[['@reboot](mailto:'@reboot)', ['@yearly](mailto:'@yearly)', ['@annually](mailto:'@annually)', ['@monthly](mailto:'@monthly)', ['@weekly](mailto:'@weekly)', ['@daily](mailto:'@daily)', ['@midnight](mailto:'@midnight)', ['@hourly](mailto:'@hourly)'] |
| [`load_config`](_build/airflow_cron.load_config.md#airflow_cron.load_config)(path)                                    |                                                                                                                                                                                                                                                                            |
| [`load_airflow_config`](_build/airflow_cron.load_airflow_config.md#airflow_cron.load_airflow_config)(path)            |                                                                                                                                                                                                                                                                            |
